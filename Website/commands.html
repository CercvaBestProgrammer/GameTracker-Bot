Command Description Usage
>info Get information about server. >info [serverip], also you can use it when you had set name on ip.
>rank See server rank. >rank [serverip], also you can use it when you had set name on ip.
>maps Get information which maps are used frequently. >maps [serverip], also you can use it when you had set name on ip.
>top10 See Top 10 players of the server. >top10 [serverip], also you can use it when you had set name on ip.
>setname With this command you can name the IP and then search easily. >setname [serverip] [name]
>queue See all ips and names >queue
>clear clear queue. >clear
>playerinfo Get player statistics. >playerinfo [ip] [playername] also you can use it when you had set name on ip
>playerscore Get player score. >playerscore [ip] [playername] also you can use it when you had set name on ip
>playertime Get playertime statistics. >playertime [ip] [playername] also you can use it when you had set name on ip
>vars See Server variables. >vars [serverip], also you can use it when you had set name on ip.
